pechnata hai .. toh ab git kaam kr skta hai , jsey ki wo untracked(U), tracked(A), staged, 
and saved checkpoints create kr skta hai 

git kuch interesting kaam b kr skta hai jsey ki apki kitni files kis stage pr hai wo bta skta hai

-> git initialize kro
-> check kr skte ho konsi file kis stage pr h -> git status -s
-> check kr skte ho kitne saved points hai -> git log --oneline --graph

git status sirf unka status btayga jo untracked hai yan commit nhi h yan modified , once u 
commited , after that no status has been shown to u 

git status btayega commit ke phle and bdh ke files stages 
git log ka kaam h saaere commit histories btana 

Branching commands

to create the branch -> git branch feature/file_name
to switch to that branch -> git switch branch_name
to check/see the branch -> git branch 
to merge branches we have go to the main branch then -> git switch main
then to merge -> git merge file_name
to delete the branch -> git branch -d branch_name
